# Name Picker Apps

A collection of vibe-coded name picker tools — quick, self-contained apps for randomly selecting names, whether you're picking teams, raffle winners, who goes first, or anything else that needs a fair (or fun) random choice.

🔗 **[Try them live on GitHub Pages →](https://ahmedh00.github.io/name-pickers/)**

## Why this exists

Sometimes you just need a simple tool to pick a name from a list, and building one is a fun little exercise. This repo collects those experiments in one place.

## Try it instantly (no download)

Every app is hosted via GitHub Pages, so you can click straight in and use it from your browser:

👉 **https://ahmedh00.github.io/name-pickers/**

Each app gets its own page — see the table below for direct links.

## Run it locally

Every app is a single, self-contained HTML file built with vanilla HTML/CSS/JS — no frameworks, no build steps, no dependencies to install.

```bash
git clone https://github.com/ahmedh00/name-pickers.git
cd name-pickers
```

Then just double-click any `.html` file, or open it via `File > Open` in your browser.

## Contributing

Adding a new picker:

1. Build it as a single `.html` file — vanilla JS, inline CSS, no external dependencies
2. Give it a clear, descriptive filename
3. Add a row to the Apps table above
4. Open a PR

## Setting up GitHub Pages

If you haven't enabled this yet: go to **Settings > Pages**, set the source branch to `main` (or `gh-pages`) and root folder, save, and your apps will be live at the URL above within a minute or two.
